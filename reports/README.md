# Звіти

У цій папці зберігаються звіти до лабораторних робіт у форматі Markdown.

Кожен звіт має бути написаний українською мовою та містити повне оформлення, яке вимагається в PDF із завданням:

- титульну сторінку з темою роботи, автором, курсом навчання, групою та викладачем;
- постановку задачі;
- мету роботи;
- опис реалізації;
- програмний код з поясненнями;
- тестування;
- команди запуску;
- висновки.

Наявні звіти:

- `lab01.md` - статистичний аналіз тексту;
- `lab02.md` - конвертер величин;
- `lab03.md` - гра "Життя";
- `lab04.md` - гра "Криниця-Ножиці-Папір";
- `lab05.md` - гра в монети;
- `lab06.md` - симуляція роботи банкомату;
- `lab07.md` - гра "П'ять в ряд";
- `lab08.md` - гра "Морський бій".

Перед здачею потрібно заповнити в титульних сторінках фактичний курс навчання, номер групи та ПІБ викладача, якщо ці дані не були відомі під час підготовки звітів.
