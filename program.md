# Java labs

This is work to implement all assignments in labs.

## Preparation

Before implementing labs read all lectures inside ./lectures folder and create ./lectures/lectures.md file describing key points in each lecture so you can navigate to a lecture if that's a requriment. Also pinout dependencies which are taugh in lecture, you will use them in lab works.

## Implementation

Then start implementing lab assignments. Here are steps you need to do:

1. Create lab{number} git branch for the lab
2. Create lab{number} maven submodule for the lab
3. Read lab PDF file in ./labs to understand a task
4. Write assingment code in appropriate module
5. Compile and run the lab with 10 seconds timeout
6. Write tests and run them to confirm correctness of written code
7. Create report in Ukraining language for lab in ./reports folder in markdown file

Once all these tasks successfully completed then the lab can be marked as done in README.md table.
